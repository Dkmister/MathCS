O projeto

Este site foi criado com o propósito de estabelecer o conteúdo básico em matemática para a próxima geração de estudantes de ciência da computação. Estaremos trazendo conteúdos e cursos sem fins lucrativos para futuros estudantes de computação.

Por quê?

Existe uma enorme defasagem em diferentes universidades por parte dos estudantes de computação. Inúmeras reprovações em Cálculo, Álgebra, Estatística, Matemática Discreta, Lógica. Por este motivo a criação deste website foi necessária, para auxiliar os estudantes a terem os conhecimentos que, muitas vezes, não são ensinados no ensino médio e fundamental.

MathCs, ou Matemática Fundamental para Ciência da Computação, busca da melhor forma cobrir essas defasagem, disponibilizando materiais multimídias feito por estudantes do curso de bacharelado em Ciência da Computação.

Como ajudar?

Você pode ajudar entrando em contato comigo, via Contato, na barra navegacional. Ou entrar via Github e olhar o projeto detalhado via a barra Projeto Estrutrado, disponibilizado no site. A escolha é sua!
